# PokeWalk A spoofing software that turns your keyboard into a physical spoofing device. Rooted android device needed. Recommended to use with Pokemon GO, Pokemod, PGTools.

Requirements:

keyboard
adbutils
threading

Connect your phone via a USB cable. Open the App Ninjas GPS Joystick app, click start and then you can use your keyboard as a spoofing device.

Commands

Use arrow keys to walk.
s 30
Sets speed to 30 km/h
gpx [file path]
Uses a GPX file to simulate a route
Arrow keys: use arrow keys to walk.
WASD: use WASD for walking with double speed.
lat, lon: copy paste coordinates and press enter to teleport.
